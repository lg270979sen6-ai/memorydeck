🐘 Импорт из Anki
Файл .apkg (экспорт из Anki Desktop)
Как экспортировать из Anki
1
Откройте Anki Desktop на компьютере
2
Меню Файл → Экспорт
3
Формат: Anki Deck Package (.apkg)
4
Перенесите файл на телефон (AirDrop, iCloud, email)
✅ Импортируется: текст карточек, прогресс повторений (интервалы, ease factor), картинки ≤ 200 КБ, аудио ≤ 500 КБ (mp3/ogg/wav), теги, структура колод.
⚠️ Не импортируется: аудио > 500 КБ, картинки > 200 КБ, LaTeX-формулы. Аудио воспроизводится автоматически при показе карточки.
